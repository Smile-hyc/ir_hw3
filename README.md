# 中文课程资料混合检索与查询增强系统

## 功能特点

- 读取本地中文课程资料文件。
- 按段落切分文本片段 chunk。
- 使用 `jieba` 进行中文分词，并过滤简单停用词。
- 支持 BM25 关键词检索。
- 支持 TF-IDF 向量空间检索。
- 支持 BM25 与 TF-IDF 加权融合的混合检索。
- 支持 BM25 权重调节和 Top-K 结果数量选择。
- 支持错别字纠错提示，不自动替换用户查询。
- 支持课程术语查询扩展，可选择使用扩展后的查询检索。
- 展示原始查询、查询分词、扩展词、实际检索查询、命中关键词高亮和检索分数解释。

```bash

```

运行后在浏览器中打开 Streamlit 提供的本地地址即可使用。

## 查询增强模块

本项目在基础混合检索之外加入了轻量级查询增强模块，主要包括错别字纠错和查询扩展两部分。错别字纠错通过课程术语词典和字符串相似度计算，为用户输入中的近似错误提供建议；查询扩展通过预设相关词表，为用户查询补充领域相关词，从而缓解查询过短或表达不完整的问题。该模块不依赖大模型，完全基于本地规则实现，适合作为信息检索课程作业中的可解释 demo。

## 设计亮点

1. 轻量级实现，无需大模型。
2. 支持中文分词与中文课程资料。
3. 对比 BM25、TF-IDF、Hybrid 三种检索方式。
4. 支持错别字纠错提示。
5. 支持课程术语查询扩展。
6. 展示分数、命中词和实际检索查询，具有可解释性。
