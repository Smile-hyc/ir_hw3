# 向量空间模型与 TF-IDF

向量空间模型把查询和文档都表示为向量。向量中的每一维通常对应一个词项，维度上的数值表示该词项在文本中的重要程度。这样，文本相关性就可以转化为向量之间相似度的计算问题。

TF-IDF 是向量空间模型中常见的词项权重方法。TF 表示词频，用来描述词项在当前文档中的出现情况；IDF 表示逆文档频率，用来衡量词项在整个语料中的区分能力。二者结合后，可以降低常见词的权重，同时提高重要课程术语的权重。

余弦相似度常用于比较查询向量和文档向量的方向是否接近。即使两个文本长度不同，只要它们在关键词分布上相似，余弦相似度也可能较高。因此，TF-IDF 检索适合发现整体主题相关的课程资料片段。

在本 demo 中，系统使用中文分词后的词项构建 TF-IDF 矩阵。当用户输入查询时，查询也会被转换为同一词表下的向量，并与每个文本片段计算相似度，从而得到向量空间模型下的排序结果。
