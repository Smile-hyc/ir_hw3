# 查询扩展与查询增强

查询扩展是一种改善检索效果的方法。用户输入的查询往往很短，可能只包含少量关键词，系统可以根据同义词、相关术语或上下文信息补充更多表达方式，从而提高召回率。例如，“倒排表”和“倒排索引”在很多课程语境中关系密切。

查询增强还可以包括错别字纠正。用户可能输入“倒排锁引”“向量摸型”或“余弦相似渡”这样的错误写法，系统可以根据课程术语词典和编辑距离给出建议。即使不自动替换查询，提示“你是否想搜索某个术语”也能帮助用户修正查询意图。

在信息检索系统中，理解用户查询意图非常重要。同一个概念可能有不同说法，不同用户也可能使用不同表达。通过维护课程术语词典，系统可以把查询和课程知识点联系起来，使检索过程更贴近学习场景。

查询扩展需要注意平衡召回率和准确率。如果扩展词过多，系统可能返回大量主题相近但并不精确的结果；如果完全不扩展，又可能漏掉相关资料。本 demo 采用轻量方式，只展示分词、纠错建议和关键词高亮，便于理解基本思想。
